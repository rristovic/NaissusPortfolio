Polyclinic GOG

Project description

The project is a simple web-portal for presenting polyclinic's available services and information. It ulitizes bootstrap toolkit to build beautiful and intuitive UI.

Contains a section on the main page where weekly and daily news is presented, a section for viewing all available services, all available doctors and polyclinic's history. The last page is a contact page, where one can directly contact doctor currently on his duty through simple contact form.

This project was built using Bootstrap framework and a simple servlet on the backend written in java, spring. The news is published and checked regulary every day, composed of data stored on the backend, where administrator can easily add more data. Servlet is also used for the quick contact form.

  • Type: Web application
  • Technologies: jQuery, Bootstrap, Java, Spring, Postgress, Tomcat
  • Status: Shutdown
  • Date: January 2013.

World Of GameOfThrones

Project description

The project is an e-commerce and blog web portal for the sale of specific themed items (in this case: "Game of thrones").

Contains a home page, where daily blogs are listed, and shop page where the user can browse and buy various items. The user can log in, fill the details only once, and can review its viewed and bought items later. The user can connect its social accounts and also share posts on social platforms. Lastly, there is a standard contact page.

This project was built using a WordPress framework. WooComerce toolkit was used for selling items, sending invoices and managing all online commerce. Many third parties plugins were used. For example, WooCommerce EU VAT Compliance, WP SuperCache for caching php generated pages, WP Optimizer for managing and optimising the database, various spam and broken links checkers, beautiful slider builder plugins, etc.

  • Type: Web application
  • Technologies: JavaScript, jQuery, PHP, WordPress, MySQL
  • Status: Shutdown
  • Date: May 2014.

Zar - food order

Project description

The project is a web application used for company internal food ordering. The application contains two views. One is the user's view and another, manager's point of view.

User's view contains a home page where an employee can see available meals, its current order status, and its order history. A user can search through all meals via search input field, can make order variations by selecting different types of packaging, different salads and aperitives. The user can delete its current order, or repeat its last order by selecting from the order history.

On manager's view, there is a list of today's orders made by company employees, listed by the date and user information. A manager can print orders for the ongoing day in pure text or excel formatting. The manager can also contact the user if some problem has emerged.

This project was built upon JavaScript MVC frameworks. Backbone.js and Marionette.js were main libraries used for supporting this project. The backend is written in Java, Spring and Hibernate frameworks, supported by PostgreSQL database.

  • Type: Web application
  • Technologies: JavaScript, Bootstrap, Backbone.js, Marionette.js, Java, Spring, Hibernate, Postgress, Tomcat
  • Status: Active
  • Date: September 2016.

Interfon - magazine app

Project description

The project is an android application created specifically for the magazine Casopis InterFON based in Serbia. It is simple news reader application.

Consists of the main view, where the user can view all available articles. The user is presented with tabs, which represent magazine article categories, from which it can swipe and chose desired article category to go through. The user can save articles for later, share an article, rate and comment it.

This project was built in Android Studio using plain Java language. Third party libraries were used like Volley, GR EventBus, Picasso, UIL, Dagger, etc. The backend was developed in WordPress, with a custom plugin for REST APIs.

  • Type: Android application
  • Technologies: Android application
  • Technologies: Java, Android, XML, SQLite, Volley, 3rd party libraries.., Google Cloud Platform, PHP Plugin Dev
  • Status: Active
  • Date: January 2017.

Happy Dino - arcade game

Project description

The project is an android arcade game built without using android layout libraries.

Consists of the main screen where a user can continue to the game screen, or go and see the highest scores avaialble.

The game was built based on famous "Clappy Bird" game model. Its main goal was to present the simple and optimised game experience, where the user can focus on challenging a friend in a smooth environment with relaxing music.

This project was built in Android Studio using plain Java language. Google Play Services were used to synchronize game scores with the cloud, and Facebook SDK to share scores on social media. No other third party libraries were used, the game was mainly built upon "android.graphics" package.

  • Type: Android application
  • Technologies: Java, Android, XML, Google Play Services
  • Status: Active
  • Date: March 2016.

FONIS

Project description

The project is an android application created specifically for the information organization FONIS based in Serbia.

When a user enters the application, it is presented with dashboard main screen. The dashboard is UI component with four main options to chose from ("News", "MSDNAA", "About", "Projects").

The news view is a primary view which contains news from the organization website. The MSNDAA view is a place where the user can register its Microsoft account within the organization by filling out the application form and submitting required pictures. The project view contains organization's projects and for the last, there is standard about page.

This project was built in Android Studio using plain Java language. Third party libraries were used like Volley, GR EventBus, Picasso, UIL, Dagger, etc. Java Client Mail API was utilized for sending and submitting the application form for the organization account.

  • Type: Android application
  • Technologies: Java, Android, XML, Google Play Services
  • Status: Active
  • Date: February 2016.
